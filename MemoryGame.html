Memory Game 3 Day Project

This was a 3 day project, where I designed a simple memory game in javascript. The theme was about cat species, featuring lions, tigers, panthers, etc. The github of the project can be found here: https://github.com/sergiogutierrez2/MemoryKittyGame

If you would like to play this simple game, please go to the following link: Memory kitty Game

It must be noted that no effort was put into making this game look aesthetically good. It was intended to be a simple, quick, classical memory game.
